# Home page Figma asset slots

Export the Figma home layers at 2x and place them here with these names:

- `home-bg@2x.png`: full home page background, excluding interactive HTML text inputs if possible.
- `comp@2x.png`: the `home / Comp` orb artwork.
- `toolbar@2x.png`: the toolbar container behind the resume textarea and frame4 controls.
- `frame4-frame1-upload@2x.png`: `toolbar / frame4 / frame1`, upload resume button artwork.
- `frame4-group1-continue@2x.png`: `toolbar / frame4 / group1`, continue button artwork.

The app keeps CSS fallback shapes when an image is missing, so these files can be added incrementally.
